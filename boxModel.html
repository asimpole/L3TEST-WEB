CSS Box Model

The CSS box model is a container which possesses multiple properties. These include: borders, margins, padding and content.

Properties:

  • Border Area: Area inbetween the padding and margin.
  • Margin Area: This is the space inbetween the border and margin. Useful to seperate the element from it's neighbours.
  • Padding Area: The area surrounding the content, inside the border box.
  • Content Area: Area of content. Text, images and other media content. It's boundary is the edge of the content.

(GeeksForGeeks, 2018)
