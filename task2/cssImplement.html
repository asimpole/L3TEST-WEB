There are 3 different methods of implementing CSS: internal, external and inline.

Internal CSS, referred to as embedded on this page, is a useful method of styling a single page. Using internal CSS on multiple pages can be time consuming and awkward as the code needs to be copy+pasted on each page and any changes need to be manually done on every page it is present on.

External CSS is using a .css file which is referenced or imported via the "link rel="stylesheet"..." code. This is useful for using styling on a large website with multiple pages as the styling can be changed from one file which affects the whole website or group of pages. It can also improve loading times as the large CSS content is cached and upon loading one page, it never needs loading again on others.

Inline CSS is used to style a specific element. It isn't generally recommended as each tag needs individual styling however if only a single element needs styling then it makes sense.
