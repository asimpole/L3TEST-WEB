The following code is what creates this box and hover action: 
div.box {
    padding: 50px;
    border-style:ridge;
    width:200px;
    background-color: red;
}

div.box:hover {
    animation-name: hoverBox;
    animation-duration: 2s;
    animation-fill-mode: both;
}

@keyframes hoverBox {
    from {background-color: red; transform:rotateX(0deg);}
    to {background-color: yellow; transform:rotateX(50deg);}
}
The animation keyframes are defined as a type of preset which can be applied to an element via the "animation-name" variable.
